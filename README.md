# APDATIKNAS — Asosiasi Praktisi Data TIK Nasional

Portal web resmi **APDATIKNAS** — sebuah asosiasi profesional para praktisi data dan teknologi informasi nasional di Indonesia. Dibangun dengan pendekatan desain minimalis premium yang terinspirasi dari filosofi Apple Design Language.

---

## Tentang Proyek

APDATIKNAS Portal adalah situs web asosiasi yang dirancang untuk menjadi pusat informasi, komunitas, dan konten bagi para praktisi data TIK di Indonesia. Portal ini menggabungkan estetika premium tingkat tinggi dengan alur konten yang terhubung secara visual melalui animasi *scroll*.

---

## Navigasi Utama

| Menu | Deskripsi |
|---|---|
| **Heart** | Visi, Misi, Kepemimpinan, Struktur, Lokasi, dan Sejarah asosiasi |
| **Notes** | Tech Blog, Design Thinking, Arsitektur Teknologi, dan Pembaruan AI |
| **Horizon** | Podcast, Webinar, Live Event, dan Streaming YouTube |
| **Circle** | Keanggotaan, Mitra, dan Jaringan Profesional |
| **Reach** | Dukungan, FAQ, Forum Komunitas, dan Kontak |

---

## Filosofi Desain

- **Minimalis Ekstrim** — White space yang luas dan terstruktur
- **Scroll-Connected Flow** — Garis animasi abu-abu yang menghubungkan setiap bagian secara visual
- **Jargon Emosional** — *Start with Heart · Crafting Tomorrow's Legacy · The Horizon of Possible*
- **Tipografi Premium** — SF Pro Display dengan bobot Light/Extra Light dan tracking lebar
- **Dark Pill Buttons** — Tombol aksi kapsul hitam gaya Apple untuk *Our Heart* dan *Our Note*

---

## Tech Stack

- **Framework:** Next.js 16 (App Router, React 19, TypeScript)
- **Styling:** Tailwind CSS v4
- **Animasi:** Framer Motion (scroll-triggered animations)
- **UI Components:** shadcn/ui (Radix Primitives)
- **Icons:** Lucide React
- **Deployment:** Vercel

---

## Cara Menjalankan Secara Lokal

```bash
# Install dependensi
npm install

# Jalankan dev server
npm run dev
```

Buka [http://localhost:3000](http://localhost:3000) di browser Anda.
